# Recipe AI

AI-powered recipe recommendation platform developed as a graduation project.

## Project Structure

### recipe-ai-mobile
React Native / Expo mobile application.

### recipe-ai-backend
NestJS backend service.

### recipe-ai-python
Python-based AI service for food recognition and recipe recommendations.

## Features

- AI-powered recipe suggestions
- User authentication
- Favorite recipes
- Mobile application
- Backend API integration
- Food image analysis

## Technologies

- React Native
- Expo
- TypeScript
- NestJS
- PostgreSQL
- Python
